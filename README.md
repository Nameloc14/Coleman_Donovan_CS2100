# ByteMarket
### Donovan Coleman | CS2100 | Spring 2026

## Project Description
ByteMarket is a multi-page website about PC components, gaming gear, and build guides. It includes an e-commerce product page powered by Snipcart, a JavaScript image slideshow, a contact form, and an HTML comparison table.

## Pages
- Home - Landing page with image slideshow and navigation cards
- Components - PC parts info with collapsible details widgets and Intel vs AMD comparison
- Gaming - CSS image gallery showcasing gaming builds
- Products - Shop page with Add to Cart functionality via Snipcart
- Deals - Deal cards and PC build tier comparison table
- About - Contact form with validation

## Installation
No installation needed. Open index.html in any browser to view locally.
